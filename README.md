## Installation:

So, how easy is it to get started with BEM?  *Super easy*.

    It's as easy as...
    
    1 › git clone git://github.com/banzalik/enb-project-stub.git
    2 › cd project-stub
    3 › make

    (hint: execute the above commands in your terminal)
